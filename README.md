## 📂 Project Setup

1. Clone the repository

```
git clone https://github.com/ganeshtadvi/World-Country-Details.git
```

2. Go to project folder

```
cd World-Country-Details
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm run dev
```
